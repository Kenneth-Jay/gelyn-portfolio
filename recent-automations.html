Automated Lead Enrichment & Prioritization

Problem: Sales teams spent too much time manually researching leads and categorizing them, which slowed down response times and impacted follow-ups.

Solution: Built a Zapier-powered automation that:

  • Captures lead entries via webhook.
  • Extracts website domain from the lead and enriches data using Abstract API.
  • Routes leads onto separate paths based on enrichment results:
    • Startup: Companies < 25 employees and < 5 years old.
    • Established Business: Companies > 100 employees and > 5 years old.
    • Fallback: Any lead that doesn’t match the above criteria.
  • For each path:
    • Startup → Insert into Supabase with low priority tag.
    • Established → Insert into Supabase with high priority, notify sales via Slack, and create Gmail draft.
    • Fallback → Insert into Supabase with medium priority.

Result: Lead processing is fully automated, prioritized, and ensures the sales team acts on high-value leads first.

Tools Used:
Webhook (Lead Entry)
Zapier
Abstract API (Lead Enrichment)
Supabase
Slack
Gmail

Form Data Validation & Event Registration Automation

Problem: Manual verification of Tally form submissions and event registrations in Google Sheets was time-consuming and error-prone.

Solution: Built an automation in Make that:

  • Triggers when a new submission is received from Tally.
  • Reads the form details and checks if the user already exists in a Google Sheet.
  • Stores the user ID and form ID in a variable for use across multiple routes.
  • Calls a subscenario (contains proprietary API calls and db integrations.So not able to showcase what is inside it.).
  • Looks up the event using the Tally form ID in the event sheet.
  • Routes the data through an ETL process to upsert user events and registrations in Google Sheets.
  • If the user is new: Adds them to Mailchimp as a subscriber.

Result: Automated form validation, event registration, data synchronization, and subscriber onboarding — eliminating manual errors and improving efficiency.

Tools Used:
Tally
Make
Google Sheets
Mail Chimp

Email Attachment Processing & File Automation

Problem: Managing incoming email attachments manually was time-consuming and error-prone, especially when handling multiple file types.

Solution: Built a Make-powered automation that:

  • Watches for incoming emails with attachments and filters for doc, xlsx, csv, pdf files.
  • Uploads doc and xlsx files directly to Google Drive.
  • Sends other file types to Gemini for content analysis, generating descriptive filenames based on the file content.
  • Uploads Gemini-processed files to Google Drive with the new descriptive filenames.
  • Implements error handling: if Gemini fails, automatically sends an email alert to the administrator.

Result: Streamlined file processing, reduced manual work, and ensured proper naming and storage of all attachments.

Tools Used:
Gmail
Make
GDrive
Gemini AI
